## 開発者向け

Caymanテーマの詳細は[こちら](https://github.com/pages-themes/cayman)をご覧ください. 下記コマンドでビルドできます.

```
git clone https://github.com/ohno/julia_in_physics_2024.git
cd cayman-test
bundle install
bundle exec jekyll server
```
